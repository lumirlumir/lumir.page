html의 kbd 태그와 같이, 특정 문자를 꾸며줄 수 있는 태그가 또 있어?

---

HTML에서 특정 문자를 꾸며줄 수 있는 다양한 태그들이 있습니다. 그중 몇 가지를 소개하면 다음과 같습니다:

1. **`<strong>`**: 텍스트를 굵게 표시합니다.
    ```html
    <strong>굵은 텍스트</strong>
    ```

2. **`<em>`**: 텍스트를 이탤릭체로 표시하여 강조합니다.
    ```html
    <em>강조된 텍스트</em>
    ```

3. **`<code>`**: 코드 조각을 표시할 때 사용합니다.
    ```html
    <code>코드 조각</code>
    ```

4. **`<samp>`**: 컴퓨터 프로그램의 샘플 출력을 표시할 때 사용합니다.
    ```html
    <samp>샘플 출력</samp>
    ```

5. **`<var>`**: 수학 변수나 프로그램 변수 이름을 표시할 때 사용합니다.
    ```html
    <var>변수 이름</var>
    ```

6. **`<mark>`**: 형광펜으로 강조한 것처럼 텍스트를 표시합니다.
    ```html
    <mark>강조된 텍스트</mark>
    ```

7. **`<del>`**: 삭제된 텍스트를 표시합니다.
    ```html
    <del>삭제된 텍스트</del>
    ```

8. **`<ins>`**: 삽입된 텍스트를 표시합니다.
    ```html
    <ins>삽입된 텍스트</ins>
    ```

9. **`<sub>`**: 아래 첨자 텍스트를 표시합니다.
    ```html
    H<sub>2</sub>O
    ```

10. **`<sup>`**: 위 첨자 텍스트를 표시합니다.
    ```html
    E = mc<sup>2</sup>
    ```

이 태그들은 모두 텍스트를 꾸미거나 강조하는 데 사용될 수 있습니다. `kbd` 태그와 비슷하게, 특정 용도로 사용되는 텍스트를 꾸미기 위해 HTML에서 제공하는 다양한 태그들을 활용할 수 있습니다.
