#. 외부 주소를 통한 추가.
  - <script src="외부주소"></script>
  - <script defer src="외부주소"></script> : defer 속성을 사용할 수 있다.

#. 객체 순회
  - key : for(var key in coworkers) {console.log(key);}
  - val : for(var key in coworkers) {console.log(coworkers[key]);}

#. ES6 Three Dots
  - ...(점점점)은 'Three Dots'라고 부른다.
  - Rest Parameter와 Spread Operator의 2가지 용도로 사용한다.

#. 리터럴(literal)
  - 리터럴은 직접 표현되는 값 그 자체를 의미합니다.
  - 소스코드 안에서 직접 만들어 낸 상수 값 자체를 말하며 값을 구성하는 최소 단위

#. 숫자(number)
  - 또한, 매우 큰 수나 매우 작은 수를 표현할 경우에는 e 표기법을 사용할 수 있습니다.
  - var thirdNum = 10e6;   // 10000000
  - var fourthNum = 10e-6; // 0.00001
